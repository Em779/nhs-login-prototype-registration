Create an NHS login

To create an NHS login you will need around 20 minutes of your time. This may take longer if you are using assistive technologies.

  • A phone with signal
  • An email address
  • To prove who you are. You can do this through:
    • Information:
      Option 1. Photo ID. You will need one of the following:
      • Passport
      • UK driving licence
      • European driving licence
      • European national identity card
      Option 2. Details from your GP practice. These are:
      • Linkage key (could be called a Passphrase)
      • O.D.S. Code (could be called an Organisation Code or Practice I.D.)
      • Acccount I.D.
